MISSION: Bitcoin transactions leave permanent, public traces on the blockchain. Chain analysis applies detective techniques to identify patterns, cluster addresses, and classify transaction behaviour — revealing who owns what, and how funds move between entities.

🔍 Common Input Ownership (CIOH) — When multiple inputs are spent together, they likely belong to the same wallet. This single assumption powers most chain analysis.
🔄 Change Detection — Every payment produces a "change" output returning funds to the sender. Identifying it reveals the payment amount and the sender's remaining balance.
🕵️ Behavioural Patterns — CoinJoins obscure ownership, consolidations reveal wallet maintenance, peeling chains trace fund flows. Each pattern is a fingerprint.
